Virtual Machine deployment¶

The INDIGO-DataCloud softwares are dockerized and each one run on a specific VM(Fig.1)

Fig.1: PaaS component architecture scheme

On the Openstack tenant the virtual machines that will contain the Laniakea INDIGO-DataCloud services has to be created (table.1) Default caracteristic:

  • SO = Ubuntu 16.04

  • Open ports = 22,80
